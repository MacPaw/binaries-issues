## Description

macOS Binaries is a public collection and a knowledge base for macOS system binaries, executables, and frameworks. 

All binaries are organized by their paths. 
Every single entry in the collection contains an **overview**, a **description**, and a **source** field.

The information is accessible through search.

## How to contribute

To add a new binary entry to the database, [create](https://github.com/MacPaw/binaries-issues/issues/new?assignees=&labels=&template=add-new-binary.md&title=Add+%22Binary+Name%22) an issue by filling out the template. Once we verify your suggestion, it will appear in the database.
