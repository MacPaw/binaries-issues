## Description

macOS Binaries - is the public collection and knowledge base for macOS system binaries, executables, and frameworks. 

All the binaries are organized by their paths. 
Single entry contains an **overview**, a **description** and a **source**.

The information is accessible with search.

## How to contribute

To add new binary entry to the database, [create](https://github.com/MacPaw/binaries-issues/issues/new?assignees=&labels=&template=add-new-binary.md&title=Add+%22Binary+Name%22) issue by filling-out the template. Once we verify the addition, it will be added to the database.
